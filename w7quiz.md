# Week 7 Quiz

1. **Why is Amazon EC2 categorized as Infrastructure as a Service (IaaS)?**
   - **Answer:** B) It provides raw, configurable fundamental infrastructure components like virtual machines, storage, and load balancers.

2. **Which of the following statements is true regarding EC2 User Data scripts?**
   - **Answer:** C) They run only once during the first boot of the instance and are executed by the root user.

3. **In the EC2 instance naming format "t2.micro" or "m5.2xlarge", what does the numeric digit (e.g., 2 or 5) represent?**
   - **Answer:** B) The generation of the instance released by AWS.

4. **Which EC2 instance category is most appropriate for high-performance computing (HPC), machine learning model training, and game servers?**
   - **Answer:** C) Compute Optimized

5. **If you are deploying an in-memory database or a real-time data processing application that handles massive data sets locally, which instance type should you select?**
   - **Answer:** D) Memory Optimized

6. **Security groups act as virtual firewalls at the network layer. Which type of rules can they contain?**
   - **Answer:** C) Only allow rules.

7. **What is the default traffic configuration for a newly created AWS Security Group?**
   - **Answer:** A) All inbound traffic is blocked, and all outbound traffic is allowed.

8. **To allow secure command-line terminal access to a Linux EC2 instance, which port must be open in your security group rules?**
   - **Answer:** B) Port 22 (SSH)

9. **Which administrative task does Amazon RDS handle automatically to minimize operational overhead for a development team?**
   - **Answer:** C) Applying operating system security patches and running automated backups.

10. **How does Auto Scaling operate within Amazon RDS compared to Amazon EC2?**
    - **Answer:** C) RDS auto-scaling applies exclusively to dynamic storage capacity growth, unlike EC2 which handles compute auto-scaling.
